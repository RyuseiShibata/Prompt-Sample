# コンテンツ作成・コミュニケーション支援
コンテンツの生成や、日常的なコミュニケーションを支援するプロンプトです。文章作成や校正、プレゼンテーション資料の作成に適しています。
## 文章の生成、校正、改善提案
### シチュエーション
報告書の手直し誰かにしてもらいたいなぁ…
### プロンプト
```
あなたはプロの校正者です。以下の文章について、誤字・脱字、語尾の不統一のみを修正したうえで、普通の文章形式で出力してください。
その後、どの箇所を修正したかを表形式で教えてください。

"""
結論
本報告書に基づき、新しい人材育成プログラム「HR-2023」実施に際して、以下の主要結果を確認しました。

教育の深化：新たなトレーニング手法やオンラインツールの採用により、社員のスキル習得速度が25%向上しました。これにより、年間の生産性向上が約3万時間に達してます。

従業員の満足度：最新のフィードバックシステムの導入により、従業員の離職率が50%減少。社員の満足度は5%上昇してます。

人材の流動性：新しいキャリアパス設計により、社内の人材が活性化。特に、マネジメント層での昇進が40%増加してます。

以上の結果から、プログラム「HR-2023」は成功で、今後のさらなる拡大教育が必要であると提案します。次の手順として、研修部門と採用部門の協力を強化し、新たな人材の獲得や既存社員のスキル向上に焦点を当てる方針を立てることに決定しました。最後に、プロジェクトチーム全員の努力に感謝し、今後も引き続き、先進的な取り組みにより、当社の人材戦略と社内の活性化を進める方針に従って参ります。
"""

「しりてが」接続詞は使わず、「にもかかわらず」（逆接）や「ことにより」「ために」（因果関係）のような表現（= ロジカル接続詞）を使い、前後の文章の関係を明確にした上で、形式はそのままで文章を作り直してください。
```
## プレゼンテーション資料の作成支援
### シチュエーション
パワポのシナリオがなかなか思いつかないなぁ…
### プロンプト
```
あなたは大手IT企業でエバンジェリストとしても活躍した、プレゼンテーション資料作成のプロフェッショナルです。
以下の【制約条件】に従って、魅力的なプレゼンテーションの構成案の作成を行ってください。

【制約条件】
以下のプレゼン内容の前提条件に基づき、構成例を参考に出力を行ってください。
以下の【PREP法】を意識して出力を行ってください。ただし、1枚のスライドにPREPの要素を全て盛り込むのではなく、全体の流れを意識してください。
##【PREP法】
Point：要点（結論・主張）
Reason：理由（結論にいたった理由・そう主張する理由）
Example：具体例（理由に説得力を持たせるための事例・データ・状況）
Point：要点（結論・主張） まず初めに「要点」を伝え、次に結論に至った「理由」を説明します。そのあと理由に納得してもらうための「具体例」を提示し、そして再び「要点」を伝える、という文章構成のことです。

##【プレゼン内容の前提条件】
テーマ：「次世代の人材育成戦略」
何のために（WHY）：社内での新しい育成手法の特徴と利点を伝え、全社員の理解と協力を促す
どこで（Where）：社内で
いつ（When）：全社員向け
誰が/誰に（WHO）：新しい戦略による人材育成の方針変更に疑問や不安を持っている社員に
何を（WHAT）：新しい研修プログラムの概要、メンターリング制度の導入とそのメリット、キャリアパス設計の見直しと目標
どのようにして（HOW）：ストーリーテリング形式
結果としてどうして/どうなって欲しいか（GOAL）：新しい人材育成戦略が効果を発揮し、組織全体のパフォーマンスと生産性の向上に貢献する。

スライド枚数：10枚
時間制限：15分

## 構成例
<1枚目> タイトルスライド
<2枚目>
<3枚目>
...
```
